# 网页截屏工具

一个简洁的在线网页截图工具，基于 html2canvas 开发。

## 功能

- 📸 输入网址即可截图
- 🎨 支持自定义截图区域
- 💾 一键下载保存
- 🔄 实时预览

## 使用

1. 在输入框中输入要截图的网页 URL
2. 点击"截取屏幕"按钮
3. 截图完成后可预览并下载

## 技术栈

- HTML5 + CSS3
- JavaScript (ES6+)
- [html2canvas](https://html2canvas.hertzen.com/) - 网页截图库

## 在线演示

部署到 Vercel/Netlify/Cloudflare Pages 后访问即可使用。
